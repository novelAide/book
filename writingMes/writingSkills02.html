# 二、爽文核心


从宏观角度而言，爽文无非就是撩拨人们心中的欲望，具体的感受主要是以下三种：


# 畅快感


所谓的畅快，简单说就是随心所欲，自由自我，主要由暴力与性爱获得。经典的做法有：


  1. 主角碾压反派。
  2. 破坏规则，挑战权威，满足叛逆心理。
  3. 床戏：满足生殖冲动。
  4. 狠削装B犯/傲娇女，满足现实中的仇富心态。
  5. 迷宫夺宝，满足红果果的占有欲。
  6. 胖揍狗腿子。
  7. 战斗

# 成就感


所谓的成就，就是实现自我价值，以及由此产生的自豪，主要由克服重重困难获得。主角在逆境中发掘自己的优势，实力渐渐提升。读者因此获得人物养成的朴素的成就感。（为了增强感度，这优势一般不怎么起眼）这里所说的实力，可以是个人武力，也可以是受主角支配的人力，财力，物力。前者的集中表现形式，就是所谓的"升级"；而后者的集中表现形式，则叫做"种田"。经典的做法有：


  1. 推倒冰山美女
  2. 干翻BOSS3, 制造出某神器
  3. 调教

# 优越感


所谓的优越感，就是高高在上，受人膜拜的虚荣，主要靠装B获得。如果说畅快感的关键词是"爆发"，成就感的关键词是"稳定"，那么优越感的关键词就是"含蓄"。经典的做法有：


  1. 有眼不识泰山
  2. 人之患在好为人师
  3. 别人眼中的宝物，主角"这些垃圾，老子赏你了"。

但是，道理你尽管懂，可如何表现出来后让你感同身受被带着走，这就需要对故事铺垫、伏笔、节奏等方面的把握了，就好比相声，台词谁都会，有的人讲就能哄堂大笑，有的人讲则是寡然无味。有的小说尽管也打脸，但看着就像一出小学生作文似的闹剧，尽管也装逼，但看着就好像成功爽文里装逼被打脸的反派。而这，才是区别真正的小说大神与码字工的所在。

# 三、爽文心理学


代入感与心流的关系：心流是一种十分专注和兴奋的心理体验。当我们在看小说时所产生的代入感，除了激素作用，也有心流的作用。要让读者达到心流状态，一个很重要的因素就是主角能力与任务的挑战性的匹配度。



如果水平太低，而挑战又太困难，主角和读者会感到焦虑，一般虐主文都有点这个意思，而作者强行用金手指让主角开挂，也会被读者诟病不真实。


而如果主角已经很厉害，却还是在打小杂鱼，读者就会觉得无聊：打菜鸡有什么意思？


只有技能与挑战相匹配，主角拿下一个挑战，学会一个技能，再拿下更大的挑战，才能让读者看得很流畅、很爽。所以大部分小说都遵循「打怪升级」的模式，就算是再怎么大开金手指，也会让主角在历练中成长。


流畅度高，情绪代入越容易。


如果你有留意过爽文的文笔，会发现他们并不讲究文学性、艺术性，甚至成语都很少用几个，在阅读上绝对不会给你造成任何的麻烦。


但很大一部分网文写手，都有扎实的文字功底。换句话说，他们本来有80分的实力，但在网文中只体现出60分的水平。


比起复杂的文字，简单的文字的确会让人更爽一点。这是因为人加工信息的流畅性会对人的情绪产生影响。



流畅性—情绪联结模型认为，高加工流畅性能够引发积极情绪，虽然这种情绪很短暂、微弱，但却成为后续认知评价判断的重要线索，从而导致个体产生更为积极的评价。


我们在看小说时，文笔越是简单利落，加工流畅性就越好，就越容易追着看下去。


因此，爽文一般都干脆利落，绝不会堆砌各种复杂句子为难读者。


公众号们也谙熟这一秘密，所以你可以观察到，公众号的文章基本上都是多短句、多分段，生怕句子一长会让读者产生阻滞感。


所以说，爽文的爽点固然千变万化，套路也在推陈出新，但万变不离其宗。而它们的「宗」，可能就是这三点：题材对味，剧情循序渐进，文笔干脆利落。


# 四、爽文与期待感


所谓的"爽"，无非就是四个字——满足期望。



当然，以上套路只是最基本的，作者为了增强爽快感，一般会有两种操作：


第一种是，铺垫时加强读者的期望。



第二种是，在结尾处超额满足读者的期望。



除了上面这些主角主动的，还有被动满足了读者期望的。



不单单是网文，金庸的武侠小说也同样深谙此道，比如《倚天屠龙记》中大战光明顶、赵敏抢婚都是符合读者期望的剧情。



这就是"爽"，作者不断制造冲突，引起读者对某一剧情方向的期望，然后满足读者的期望，在期望满足的同时埋上冲突，勾起读者下一个期望，不断循环。


# 五、爽文段子合集


1. 【直入主题，绝不拖泥带水，主角全方位秒天秒地】


开篇不废话，上来便开挂。 或者穿异世，也许鬼专家。 可能雇佣军，更甚特种兵。 医术惊鬼神，武术更强大。 扮猪吃老虎，打脸啪啪啪。 对面高手屌，起脚便秒杀。


2. 【色方面】


登场必救女，翘臀xx大。 文中各女角，统统没争差。 风流俏寡妇，母女皆似花。 最美女杀手，晕倒在我家。 隔壁小少妇，短裙黑丝袜。 xx女房东，当面xxx。 虽是学生妹，身材xx佳。 休提遇xx，必是女x花。 还有小护士，绝x有一发。


3. 【财、物方面】


钱财千万贯，只是没处花。 公司和集团，粗计数百家。 天气已转凉，王氏破产啦。 主角手一挥，股市瞬间垮。 偶得空间戒，种田顶呱呱。 不但周期短，而且大如瓜。 拿到市场买，立时成天价。 我抢你的宝，全因我强大。 敢动我的草，灭光你全家。 废品不起眼，后必震天下。 各路神仙抢，最后全白瞎。


4.【战斗】


挥手灭星球，抬脚震宇宙。 招式龙傲天，众人惊成狗。 武器闪红光，飞星大如斗。 无论怪多强，三回即败走。 还有某天尊，更有某神王。 单凭主角怒，立时哔了狗。 金庸太啰嗦，古龙徒造势。 哪有我的强，全凭一挥手。


5. 【求不得】


武侠文高人秘籍必备，仙侠文财侣法地齐全。

争霸文豪杰磕头就拜，历史文诗词信手拈来。

穿越文百科自在心中，游戏文神器运势逆天。

后宫文御萝贵妓皆纳，官场文领导言听计从。
